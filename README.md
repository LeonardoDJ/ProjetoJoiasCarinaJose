# 💍 Joias & Acessórios — Carina Delfino Pivetta José
 
Site de vitrine e vendas para revendedora de joias e acessórios femininos. Desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem backend, sem cadastro de clientes.
 
---
 
## ✨ Funcionalidades
 
- **Página inicia** com apresentação elegante e chamada para ação
- **Seção Sobre** com foto, texto de apresentação e estatísticas
- **Catálogo de produtos** separado por categorias com filtro dinâmico
- **Botão de pedido via WhatsApp** em cada produto
- **Botão via Instagram** para ficar por dentro 
- **Seção de contato** com link direto para o WhatsApp
- Animações suaves de scroll reveal, partículas flutuantes e transições elegantes
- Layout totalmente responsivo (desktop e mobile)
